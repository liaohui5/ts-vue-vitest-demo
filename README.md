## 介绍

学习 unit-test 实战 demo 项目, 重点不是源码, 而是单元测试

## 安装

```sh
git clone https://github.com/liaohui5/ts-vue-vitest-demo && cd -

npm i
```

## 关于接口

接口是使用[msw.js](https://mswjs.io/)模拟的,数据交互逻辑并不是这个项目的重点,单元测试和 TDD 开发模式才是重点

## 启动

- 重点是单元测试

```sh
# 查看运行效果, 注意需要服务端
# npm run dev
npm run test
```
